# Project Blueprint

## Overview

This project is a data visualization dashboard that allows users to upload, analyze, and visualize their data. The application is built using modern web technologies, including HTML, CSS, and JavaScript, and leverages Web Components for creating reusable UI elements.

## Styles and Design

The application's design is modern, clean, and intuitive. It uses a consistent color palette, typography, and spacing to create a visually appealing and user-friendly experience.

### Colors

*   **Primary:** `#348cf0`
*   **Secondary:** `#06b6d4`
*   **Background:** `#f8fafc`
*   **Card:** `#ffffff`
*   **Text:** `#0f172a`
*   **Muted:** `#64748b`
*   **Border:** `#e5e7eb`

### Typography

*   **Font Family:** 'Inter', sans-serif
*   **Base Font Size:** 16px

### Components

*   **Buttons:** The application uses a variety of buttons, including primary, outline, and large buttons.
*   **Cards:** Cards are used to display content in a visually appealing and organized manner.
*   **Forms:** The application uses forms for user authentication and data upload.

## Features

### Current Features

*   **User Authentication:** Users can sign up and log in to the application.
*   **Data Upload:** Users can upload their data in CSV format.
*   **Data Visualization:** The application automatically generates charts from the user's data.
*   **Interactive Tables:** Users can sort, filter, and explore their datasets in an interactive table.
*   **AI-Powered Insights:** Users can ask questions in natural language to get insights from their data.
